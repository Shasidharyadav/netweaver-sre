Mission Overview
You are an autonomous SRE agent managing a simulated 100-GPU Clos network cluster. Incidents occur — node failures, buffer congestion, silent NaN contagion — and your job is to diagnose them from telemetry signals and issue the correct remediation command as quickly as possible. Speed matters: each wasted step reduces your reward. Each scenario runs for up to 10 steps. Choose a task on the left, read the telemetry, and transmit the right command.

⭐⭐⭐ Task 03 — Silent NaN Contagion
Hard
1
No node logs. Your only signal is the Gradient Variances chart (ranks 0–9). One bar will spike dramatically — that's your suspect rank index.
2
Use RUN_MINI_ITERATION with Target set to a rank range (e.g. 3-3) to confirm which rank index contains the NaN source. The log will respond with "anomaly found" or "none".
3
Once confirmed, switch to DRAIN_TRAFFIC and target the exact offline node shown in the topology grid (which is tagged as suspect after confirmation).
4
Fewer binary-search steps = higher reward. Draining the wrong node incurs a −0.10 penalty.

Scoring & Tips
reward = max(0.1, base − (step − 1) × decay)
Decay varies by task (0.15 / 0.12 / 0.13). Act on step 1 for maximum reward.

TIP: Use ↺ Reset Episode to randomise the failing node / threshold between runs.
TIP: In Hard mode, use binary search — try range 0-4 first, then narrow.
TIP: Cumulative reward accumulates across steps; a failed episode still scores if partial rewards were earned.
