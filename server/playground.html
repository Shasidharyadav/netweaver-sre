How To Play - SRE Triage

Goal: Maintain the server cluster by triaging specific faults. Read the System Logs and Telemetry to pinpoint the issue and send exact Action Commands to resolve it efficiently for maximum reward.

Easy Mode

The log tells you which node crashed.
Solution: Select DRAIN_TRAFFIC, input the exact node (e.g., node_45) into Target, and Execute.

Medium Mode

A buffer overshoot happens! The logs give a Target integer.
Solution: Select TUNE_PFC_THRESHOLD, enter the integer into Value. Execute.

Hard Mode

A NaN gradient breaks loose! The Telemetry Array will show a spike (e.g., 999.9).
Step 1: Count the index of the spike (0-15). Launch RUN_MINI_ITERATION with Target "X-X" (e.g., "7-7").
Step 2: The log confirms node_XX. Run DRAIN_TRAFFIC on that node!
